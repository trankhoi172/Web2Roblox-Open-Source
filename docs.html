🚀 Quick Start

  1. Paste your HTML and CSS into the left panel.
  2. Click the Convert button in the top right.
  3. Review the generated Roblox hierarchy in the right panel.
  4. Replace any external image URLs with Roblox Asset IDs using the Image Assets panel.
  5. Click Download .rbxmx.
  6. Open Roblox Studio, right-click StarterGui, and select Insert From File.

🖼 Images & Assets

Roblox does not support loading images directly from external web URLs (like https://example.com/image.png) for security and performance reasons.

How to fix missing images:

  1. Upload your image to Roblox using the Asset Manager or Toolbox.
  2. Copy the Asset ID (it will look like rbxassetid://123456789).
  3. In Web2Roblox, click on the missing image in the Image Assets panel.
  4. Paste the Asset ID and click Apply.
  5. Re-download the .rbxmx file.

💡 Best Practices

  • Use semantic HTML: Use <button> for clickable elements instead of <div> with click handlers.
  • Keep nesting reasonable: Deeply nested DOM trees can result in complex Roblox hierarchies.
  • Test in Studio: Always preview your GUI in Roblox Studio after importing to check for minor layout shifts.
  • Use Roblox Asset IDs: Replace all external image URLs before finalizing your project.
  • Absolute vs Relative: The tool calculates positions relative to the parent. Ensure your CSS uses standard flow or Flexbox.

❓ FAQ

Why is my icon missing?

Roblox does not support inline SVG code (like Lucide icons). The exporter replaces them with placeholder ImageLabels named Icon_[ReplaceWithAssetID]. You must upload the icon to Roblox and apply the Asset ID.

Why is my font different?

Roblox uses its own font system (e.g., Gotham, SourceSans). Custom web fonts (like Google Fonts) cannot be imported directly. The tool attempts to map font weights (Bold, Regular) to Roblox's closest equivalents.

Why is my image blank?

External URLs (HTTP/HTTPS) are blocked by Roblox. You must replace them with rbxassetid:// URLs using the Image Assets panel in the app.

Why did my layout change slightly?

Roblox's UI rendering engine differs slightly from browser engines (Chromium/WebKit). While we use the browser's exact pixel calculations, minor rounding differences or font rendering variations may occur in Studio.

Does it support JavaScript?

No. This tool converts the visual structure (HTML/CSS) only. Interactive logic (click handlers, state management) must be scripted manually in Lua inside Roblox Studio.
